# 概要

時系列データの予測を行うAI(ディープラーニング)プログラムです。

(1)単純な構造のRNN
(2)LSTM
(3)GRU

以上の3種類の学習モデルを生成するプログラムと、それらを利用してデータ予測を行う予測プログラムを公開しています。

学習・予測対象のデータセットとして、2020年7月の東京都の平均気温を取り上げています。(https://www.data.jma.go.jp/obd/stats/etrn/index.php)
